Consent Component Examples

Basic Consent Banner

Click the button below to show a consent banner:

<hot-consent consent-id="demo-site" title="Privacy Notice" message="This website uses cookies to improve your experience." agree-label="Accept All" not-agree-label="Decline"> </hot-consent>

Custom Consent Message

Custom consent with additional information:

What we collect: Usage analytics, preferences, and session data.

How we use it: To improve services and provide better user experience.
<hot-consent consent-id="custom-site" title="Enhanced Privacy Notice" message="We collect and process your data..." agree-label="I Understand & Accept" not-agree-label="I Need More Info"> <div slot="additional-info"> <p><strong>What we collect:</strong> Usage analytics...</p> </div> </hot-consent>

Tracking Component Examples

Basic Tracking Setup

Note: This is a demo configuration. Replace with your actual Matomo credentials.

<hot-tracking site-id="1" domain="demo.hotosm.org" matomo-url="https://matomo.hotosm.org" show-consent="true" consent-message="We use analytics to improve our services..."> </hot-tracking>

Tracking Without Consent Banner

For cases where consent is handled separately or not required.

<hot-tracking site-id="2" domain="demo.hotosm.org" matomo-url="https://matomo.hotosm.org" show-consent="false"> </hot-tracking>

Usage Instructions

Getting Started

  1. Include the library: Add the script and CSS files to your HTML
  2. Choose components: Use the appropriate custom elements in your HTML
  3. Configure properties: Set attributes to customize the components
  4. Handle events: Listen for component events to add interactivity

Available Components

  • <hot-header> - Navigation header with optional OSM authentication
  • <hot-logo> - HOTOSM logo with various display options
  • <hot-consent> - Privacy consent banner with customizable messaging
  • <hot-tracking> - Matomo analytics integration with consent management

Important Notes

  • Replace YOUR_CLIENT_ID and YOUR_REDIRECT_URI with your actual OSM OAuth credentials
  • Update the Matomo tracking configuration with your actual site ID and domain
  • Customize consent messages according to your privacy policy
  • Test all components thoroughly in your specific use case
